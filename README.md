# St Andrews 24/7 Taxi Service — Next.js Website

A production-ready Next.js 14 website for St Andrews Taxis, fully SEO-optimised for local search ranking.

## Tech Stack

- **Next.js 14** (App Router)
- **React 18**
- **TypeScript**
- **CSS** (global stylesheet, no Tailwind dependency)

---

## 🚀 Quick Start

### 1. Install dependencies
```bash
npm install
```

### 2. Run development server
```bash
npm run dev
```
Open [http://localhost:3000](http://localhost:3000)

### 3. Build for production
```bash
npm run build
npm start
```

## 🌐 Deploying to Vercel (Recommended — Free)

1. Push this folder to a GitHub repository
2. Go to [vercel.com](https://vercel.com) → **Add New Project**
3. Import your GitHub repo
4. Vercel auto-detects Next.js — click **Deploy**
5. Your site is live in ~2 minutes

### Custom Domain
In Vercel dashboard → **Domains** → add `standrewstaxis.co.uk`
Then update your domain DNS:
```
A record:    @    →  76.76.21.21
CNAME:       www  →  cname.vercel-dns.com
```

---

## 🌐 Deploying to Netlify (Alternative — Free)

1. Push to GitHub
2. Go to [netlify.com](https://netlify.com) → **Add new site** → **Import from Git**
3. Build command: `npm run build`
4. Publish directory: `.next`
5. Add the Netlify Next.js plugin in `netlify.toml`:

```toml
[[plugins]]
package = "@netlify/plugin-nextjs"
```

## ✏️ Key Customisations

### Update your domain
In `app/layout.tsx` and `app/sitemap.ts`, replace:
```
https://www.standrewstaxis.co.uk
```
with your actual domain.

### Update the booking webhook
In `components/BookingForm.tsx`, find:
```ts
const res = await fetch('https://kneega.app.n8n.cloud/webhook/submitform', ...
```
Replace with your own n8n or backend endpoint.

### Update email address
In `components/Contact.tsx`:
```tsx
<a href="mailto:info@standrewstaxis.co.uk">
```

### Add a favicon
Place your `favicon.ico` and `icon.png` in the `/public` folder.
Next.js automatically serves `/public/favicon.ico`.
